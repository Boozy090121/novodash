How to Run Novo Dashboard v4

This document provides instructions for setting up and running the Novo Dashboard v4 application.

Note: This application requires Node.js version 18 or later to be installed on your computer.

Option 1: Using the Batch File (Windows)

  1. Make sure you have Node.js installed
  2. Locate the start-dashboard.bat file in the root directory
  3. Double-click the batch file to run it
  4. The script will install dependencies and start the application
  5. Your browser should automatically open to http://localhost:3000

Option 2: Manual Installation and Startup

  1. Open a command prompt or terminal
  2. Navigate to the dashboard directory: 
    cd path\to\novo-dashboard-v4
  3. Install the dependencies: 
    npm install
  4. Start the development server: 
    npm start
  5. The application will open in your default browser at http://localhost:3000

Troubleshooting

If you encounter errors during installation or startup:

  • Node.js not found: Make sure Node.js is installed and in your PATH
  • Dependency errors: Try running npm cache clean --force followed by npm install
  • Port already in use: If port 3000 is already used, the application will ask to use a different port
  • TypeScript errors: Install additional types with: 
    npm install --save-dev @types/react @types/react-dom @types/node @types/react-sparklines @types/lodash
